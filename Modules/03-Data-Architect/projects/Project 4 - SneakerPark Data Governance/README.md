# Project 4 - SneakerPark Data Governance

Enterprise Data Governance and Master Data Management (MDM) solution design for SneakerPark as part of the Udacity Data Architect module.

## Project Overview
This project focuses on analysing enterprise data challenges across SneakerPark’s operational systems and designing foundational data governance capabilities to improve data quality, metadata management, and master data consistency.

## Key Areas Covered
Enterprise Conceptual Data Modelling  
Metadata & Data Catalog Design  
Data Profiling & Data Quality Analysis  
Data Quality Monitoring Dashboard  
Master Data Management (MDM) Architecture  
MDM Matching Rules  
Data Governance Roles & Responsibilities
