# Project 1 - Human Resources Database Design

Design and implementation of a normalized Human Resources relational database for Tech ABC Corp as part of the Udacity Data Architect Nanodegree.

## Project Overview
This project focuses on designing and building a secure and scalable HR database solution using relational database design principles and normalization techniques.

## Key Areas Covered
- Business & Technical Requirements Gathering
- Conceptual, Logical & Physical ERDs
- Database Normalization (3NF)
- PostgreSQL Database Design
- SQL DDL & CRUD Operations
- ETL from Staging Tables
- Data Governance & Security
- Backup & Retention Considerations

- PostgreSQL
- SQL
